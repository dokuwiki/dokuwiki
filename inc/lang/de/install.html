Diese Seite hilft Ihnen bei der Erstinstallation und Konfiguration von EasyWiki. Zusätzliche Informationen zu diesem Installationsskript finden Sie auf der entsprechenden Hilfe Seite.

EasyWiki verwendet normale Dateien für das Speichern von Wikiseiten und anderen Informationen (Bilder, Suchindizes, alte Versionen usw.). Um EasyWiki betreiben zu können, muss Schreibzugriff auf die Verzeichnisse bestehen, in denen EasyWiki diese Dateien ablegt. Dieses Installationsprogramm kann diese Rechte nicht für Sie setzen. Sie müssen dies manuell auf einer Kommando-Shell oder, falls Sie EasyWiki bei einem Fremdanbieter hosten, über FTP oder ein entsprechendes Werkzeug (z.B. cPanel) durchführen.

Dieses Skript hilft Ihnen beim ersten Einrichten des Zugangsschutzes (ACL) von EasyWiki, welcher eine Administratoranmeldung und damit Zugang zum Administrationsmenu ermöglicht. Dort können Sie dann weitere Tätigkeiten wie das Installieren von Plugins, dass Verwalten von Benutzern und das Ändern von Konfigurationseinstellungen durchführen. Das Nutzen der Zugangskontrolle ist nicht zwingend erforderlich, es erleichtert aber die Administration von EasyWiki.
